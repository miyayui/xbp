pythonでゲーム制作（2班）

＜ゲームの説明＞

2班は「タイピングゲーム」を作成した。
ランダムで出題単語が表示される。時間制限を設け、時間になったら強制的にゲームが終了する。
出題内容、単語は各々が考えた。

ゲームのソースコード

  1. import random
  2. import time
  3. # ゲームで使用する単語リスト
  4. words = ["なまむぎなまごめなまたまご","あぶりかるび","てきしゅつしゅじゅつ","すもももももももものうち","ろうにゃくなんにょ","こまだいとまこまい","てきしゅつしゅじゅつ"]
  6. def main():
  7.     print("早口言葉タイピングゲーム!")
  8.     print("制限時間は60秒！すべてひらがなで入力してね！")
  9.     input("Enterでスタート！")
  10.     
  11.     
  12.     while True:
  13.         score = 0
  14.         game_duration=60
  16.         start_time=time.time()
  17.         end_time=start_time+game_duration
  19.         incorrect = False
  21.         while time.time()<end_time:
  22.             target_word = random.choice(words)
  23.             print(f"次の言葉を打て！: {target_word}")
  25.             user_input = input()
  26.             if user_input == target_word:
  27.                     score += 1
  28.                     print("正解!")
  29.             else:
  30.                     print("不正解!")
  31.                     incorrect = True
  32.                     break # 間違えた場合、内部の無限ループを終了
  34.         if incorrect:
  35.             print(f"Game Over!:{score}問正解！")
  36.             play_again = input("もう一度挑戦しますか？（はい/いいえ）: ").lower()
  37.             if play_again != "はい":
  38.                 break
  39.         else:
  40.              break
  41.         
  42.     print(f"終了！{score}問正解！")
  44. if __name__ == "__main__":
  45.     main()
